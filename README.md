# Module-NFT

## Description

- It is a module that is used to retrieve some information about NFT from Ethereum Network 
- This module is powered by [NestJs](https://nestjs.com/) framework

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
